# BookMyStayApp
Hotel Booking Management System
Objective

The goal of this project is to build a Hotel Booking Management System using Core Java and Data Structures.
It demonstrates how data structures help manage room availability, booking requests, and prevent double booking in a real-world system.

Project Procedure

The system was developed step by step:

System Design – Created classes such as Hotel, Room, and Booking using Object-Oriented Programming.

Room Management – Rooms are stored using a HashMap for quick access and updates.

Booking System – Users can book, cancel, and view bookings.

Double Booking Prevention – The system checks room availability before confirming a booking.

Request Handling – A Queue (FIFO) is used to process booking requests fairly.

Project Flow

Initialize rooms

User requests booking

Request enters queue

System checks availability

Booking confirmed or rejected

Booking details stored

Use Cases

Add rooms

View available rooms

Book a room

Cancel a booking

View booking details

Technologies Used

Core Java

Object-Oriented Programming

Data Structures (HashMap, Queue, ArrayList)
